# Calculator

A modern web calculator with **two-line display**, **keyboard support**, and **Tailwind CSS UI**.

## Features
- **Two-line display**: shows full expression + current result
- **Keyboard support**: type numbers, operators, Delete (clear), Backspace (delete), Enter (equals)
- **Tailwind CSS UI**: modern, responsive, animated buttons
- **Basic error handling**: prevents invalid calculations
